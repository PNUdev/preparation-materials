------------------ РОЗМІТКА ТА СТРУКТУРА
<HTML> - це мова розмітки, яка використовується браузером для обробки тексту, зображень та іншого вмісту
для відображення його у необхідному форматі.

Теги в HTML: Теги - одна з найважливіших частин документа HTML. HTML використовує деякі заздалегідь
визначені теги, які повідомляють браузеру про те як відобразити його вміст.

Документ HTML в основному ділиться на дві частини: HEAD: Тут міститься інформація про документ HTML.
Наприклад, заголовок сторінки, версія HTML, метадані тощо. BODY: містить усе, що ви хочете відобразити
на веб-сторінці.

<! DOCTYPE html>: Цей тег використовується для інформування про версію HTML. На даний момент це говорить про те, що версія HTML 5.

В тегу <meta> можна визначити данні про документ HTML. Метадані - це дані про дані.
Теги <meta> завжди входять до елемента <head> і зазвичай використовуються для вказівки набору символів,
пису сторінки, ключових слів, автора документа та налаштувань області перегляду. Метадані не відображатимуться
на сторінці, але їх можна аналізувати машиною.
Метадані використовуються браузерами (як відобразити вміст або перезавантажити сторінку), пошуковими
системами (ключовими словами) та іншими веб-службами. Існує метод, який дозволяє веб-дизайнерам взяти
під контроль вікно перегляду (видима область користувача веб-сторінки) за допомогою тегу <meta>

Вікно перегляду - це видима область користувача веб-сторінки. Вона залежить від пристрою - на мобільному
телефоні він буде меншим, ніж на екрані комп’ютера.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
Данний тег дає вказівки браузеру про те, як керувати розмірами сторінки та масштабуванням.
width=device-width встановлює ширину сторінки, щоб відповідати ширині екрана пристрою (яка буде змінюватися
залежно від пристрою).
initial-scale=1.0 встановлює початковий рівень масштабування при першому завантаженні сторінки браузером.

1. НЕ використовуйте елементи великої фіксованої ширини - Наприклад, якщо зображення відображається на
ширину ширшу за область перегляду, це може призвести до горизонтальної прокрутки області перегляду.
2. НЕ дозволяйте вмісту покладатися на певну ширину області перегляду, щоб добре відображати - Оскільки
зміри екрану та ширина в пікселях CSS сильно відрізняються між пристроями, вміст не повинен покладатися
на певну ширину області перегляду, щоб добре відображати.
3. Використовуйте CSS-медіа-запити, щоб  застосувати різні стилі для малого та великого екранів.
Якщо встановити велику абсолютну ширину CSS для елементів сторінки, елемент буде занадто широким для
області перегляду на меншому пристрої.

Натомість розгляньте можливість використання відносних значень ширини, таких як width: 100%.
Крім того, будьте обережні, використовуючи великі абсолютні значення позиціонування.
Це може призвести до того, що елемент потрапляє за межі області перегляду на невеликих пристроях.


Тег <link> визначає взаємозв'язок між поточним документом та зовнішнім ресурсом.
Тег <link> найчастіше використовується для посилання на зовнішні таблиці стилів.
Елемент <link> є порожнім елементом, він містить лише атрибути.


------------ СТИЛІ
CSS розшифровується як каскадні таблиці стилів.
CSS описує спосіб відображення елементів HTML на екрані.
CSS економить багато роботи. Він може контролювати розміщення декількох веб-сторінок одночасно

HTML НІКОЛИ не призначений містити теги для форматування веб-сторінки!
HTML був створений для опису вмісту веб-сторінки, наприклад: <h1> Це заголовок </h1> <p> Це параграф. </p>
Коли теги, такі як <font>, та атрибути кольору були додані до специфікації HTML 3.2,
це створило кошмар для веб-розробників. Розробка великих веб-сайтів, де шрифти та кольорова інформація
додавались до кожної сторінки, стала довгим та дорогим процесом.
Для вирішення цієї проблеми Консорціум Всесвітньої павутини (W3C) створив CSS.
CSS видалив форматування стилю зі сторінки HTML!

Inline
Відображає елемент як вбудований елемент. Будь-які властивості висоти та ширини не матимуть ефекту.
Іншими словами, вбудовані елементи НЕ починаються з нового рядка і займають лише стільки ширини,
скільки вміст. Отже, якщо ви спробуєте встановити будь-яку ширину та висоту, це не матиме жодних ефектів.

inline-block
Відображає елемент як контейнер блоку вбудованого рівня. Ви МОЖЕТЕ встановити значення висоти та ширини.

block
Елемент розпочнеться з нового рядка і займе всю доступну ширину. І ви можете встановити значення ширини та висоти.
